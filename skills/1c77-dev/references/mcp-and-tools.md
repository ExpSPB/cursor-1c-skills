## 1. MCP 1c77-metadata

Сервер анализа `1Cv7.MD` (Compound Document + скобочный синтаксис в CP1251). Даёт API к справочникам,
документам, регистрам, перечислениям, модулям - **вместо** ручного парсинга `.mdp`/`.1s` через Python+CP1251
или слепого Grep.

- **Адрес:** `http://localhost:8009` (Project MCP).
- **Запуск:** `C:\mcp\77MCP\start.cmd` (читает `1Cv7.MD` из `C:\mcp\77MCP\data\1cv7.md`).
  Фоновый вариант - `start-hidden.vbs`. Документация сервера: https://github.com/ivanarama/77MCP.
- **Проверка доступности:** `get_configuration_info` (вернёт имя и версию загруженной конфигурации).

Инструменты (по задачам):

| Задача | Инструмент |
|--------|-----------|
| Проверить, что конфигурация загружена | `get_configuration_info` |
| Обзор объектов по типу (Справочник/Документ/Регистр/Перечисление/Журнал/Константа/Отчет/ВидРасчета) | `list_objects` |
| Реквизиты + табличные части объекта | `get_object` |
| Найти ВСЕ объекты с реквизитом по имени (трассировка использования) | `search_field` |
| Код модуля объекта (без возни с CP1251) | `get_module` |
| Глобальный модуль | `get_global_module` |
| Форма объекта | `get_form` |
| Пакетная выборка метаданных нескольких объектов | `get_objects_batch` |
| Список модулей | `list_modules` |
| Поиск по именам/синонимам/комментариям | `search` |
| Поиск текста по модулям | `search_in_modules` |
| Проверка запроса 7.7 на ошибки | `validate_query` |
| Проверка пути реквизита (`Товар.Артикул`) | `validate_field_path` |
| Сменить/перезагрузить файл конфигурации | `reload_configuration` |

Типовые сценарии:
- «Где трогается реквизит X» (миграция/рефакторинг) -> `search_field` (точнее и полнее, чем Grep по `.1s`).
- Перед написанием кода, работающего с объектом -> `get_object` (структура) + `get_module` (существующий код).
- Перед сборкой `.ert` с запросом -> `validate_query` для каждого запроса.

**Когда сервер недоступен или конфигурация не загружена** - fallback на распакованные `.mdp` + Python
CP1251 (см. п.3-4). Для очень большого глобального модуля `get_module` может быть тяжёлым - тогда точечный
Grep через `cp1251_io.py grep` по конкретному `.1s`.

## 2. Проверка качества BSL

- **`ask_1c_ai`** (MCP `1c-naparnik`) - оценка качества написанного BSL. Прогоняй после правок кода.
  Учитывай, что Напарник ориентирован на 8.x - грабли 7.7 он может не отловить; финальная истина -
  компиляция в самой 7.7 и чек-лист из `references/syntax-gotchas-77.md`.
- **`validate_query`** (MCP `1c77-metadata`) - для запросов именно 7.7 (язык запросов отличается от 8.x).

## 3. Распаковка конфигурации

Распаковать `1Cv7.MD` в дерево `.1s`/`.mdp` можно двумя путями:

- **GComp** (`-d --meta-data`, см. `references/gcomp.md`) - универсально, рекомендуется.
- **oscript + v7src** - утилита v7src на OneScript:
  ```
  oscript <путь>/v7src.os extract -bin <каталог_с_1Cv7.MD> -src <каталог_исходников>
  ```
  OneScript: `C:\Program Files\OneScript\bin\oscript.exe`. Скрипт `v7src.os` - внешняя зависимость
  (в проекте может лежать как `src/v7src.os`; если нет - использовать GComp).

## 4. Fallback: ручной CP1251

Когда MCP недоступен - читать/искать/править через Python с `encoding='cp1251'`. Это инкапсулировано в
`scripts/cp1251_io.py` (подкоманды `read`, `grep`, `replace`). Главное помнить:

- Обычный Grep по кириллице в `.1s`/`.frm`/`.mdp` **не работает** (UTF-8 vs CP1251) - не делать вывод
  «нет совпадений» по нему.
- Запись - с `newline=''`, чтобы не превратить CRLF в LF и не сломать файл для 7.7.
- Read-инструмент покажет кракозябры; для осмысленного чтения используй `cp1251_io.py read` или MCP.

## 5. Что НЕ применять к 7.7

7.7 - не 8.x. Неприменимо (это инструменты для 8.x, дадут мусор или ошибку):

- EDT-MCP (`1c-edt`) - работает с `.bsl`/`.mdo`, не с 7.7.
- `bsl-platform-help` - API платформы 8.x.
- Скиллы `1c-meta-*`, `1c-form-*`, `1c-cf*`, `1c-epf-*`, `1c-erf-*` - формат 8.x (XML, управляемые формы).
- `v8unpack` - для CF/CFE/EPF (8.x), не для `.ert`/`.md` (7.7).

Для 7.7 используем: GComp (разбор/сборка), MCP `1c77-metadata` (анализ), `cp1251_io.py` (правка),
`ask_1c_ai` (оценка, с поправкой на 8.x-уклон).
